
# Roadmap проекта

## Спринт 2: Проектирование архитектуры
- **SA/PO**:
  - Формирование технических требований.
  - Разработка C4-диаграмм архитектуры.
  - Создание дорожной карты проекта.
  - Определение критериев DoR/DoD.
- **Fullstack**:
  - Проектирование схемы базы данных.
  - Создание wireframes и пользовательских сценариев.
  - Разработка API-контрактов (OpenAPI).
- **MLOps**:
  - Настройка базовой инфраструктуры (docker-compose).
  - Настройка CI/CD пайплайнов.
  - Подготовка мониторинга.
- **AI Engineer**:
  - Анализ требований к ML-компонентам.
  - Подготовка данных и шаблонов промптов.
  - Разработка baseline-модели.

## Спринт 3:
- **SA/PO**:

  - Планирование следующего спринта.

  - Настройка трассировки LLM.
  - Проведение экспериментов.

- **MLOps**:
