# MVP scope

## Основные функции

- **Регистрация и аутентификация пользователей**
  - Профиль пользователя (вес, рост, дата рождения, пол)

- **Аналитика и отчетность**
  - Визуализация данных о сне
  - Расчет качества сна
  - Отслеживание динамики показателей
  - Генерация рекомендаций по улучшению сна

- **Интеграции**
  - Экспорт данных

## Приоритизация бэклога

### Высокий приоритет (MVP)

1. **Основной функционал**
   - Реализация основных метрик качества сна
   - Базовые графики и визуализации

2. **Профиль пользователя**
   - Завершение формы редактирования профиля
   - Валидация вводимых данных
   - Настройка уведомлений

3. **Система рекомендаций**
   - Базовые рекомендации по улучшению сна
   - Интеграция с моделью машинного обучения

### Низкий приоритет

1. **Улучшение UX/UI**
   - Адаптивный дизайн
   - Кастомизация дашборда

## Планирование спринта

- Реализация основных метрик сна
- Базовая визуализация данных
- Тестирование API
- Реализация системы рекомендаций
- Улучшение визуализаций
- Интеграция с ML-моделью
- Тестирование точности рекомендаций
- Оптимизация производительности
- Пользовательское тестирование
