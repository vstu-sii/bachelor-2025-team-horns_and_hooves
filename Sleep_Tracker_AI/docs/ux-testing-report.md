## 1. Тестирование Use Cases

### 1.1. Аутентификация
-  Регистрация нового пользователя
  - Валидация полей формы
  - Обработка дублирующихся email
  - Подтверждение регистрации
-  Вход в систему
  - Авторизация с корректными данными
  - Обработка неверных учетных данных
  - Восстановление пароля
-  Выход из системы
  - Корректное завершение сессии
  - Перенаправление на главную страницу

### 1.2. Профиль пользователя
-  Просмотр профиля
-  Редактирование личных данных
-  Обновление аватара
-  Изменение пароля

### 1.3. Управление сном
-  Импорт данных из CSV

### 1.4. Аналитика
-  Просмотр статистики сна
-  Анализ фаз сна
-  Отслеживание динамики показателей
-  Формирование отчетов
